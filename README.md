# Task Manager App

A simple Task Manager web app built using HTML, CSS, and JavaScript.

## 🚀 Features
- Add tasks
- Mark tasks as completed
- Delete tasks

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

## 📌 How to Use
1. Enter a task
2. Click Add
3. Click on task to mark complete
4. Use delete button to remove

## 🎯 Purpose
This project was built to practice JavaScript DOM manipulation and CRUD operations.
